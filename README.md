# out of Frozen
<br>

작은 진돗개의 서바이벌 퍼즐 게임<br>
<br>

Unity 2022.3.5f1 버전에서 작업되었습니다.<br>
그림소스는 모두 직접 제작하였습니다.<br>

 아래 링크를 클릭하면 파일을 다운로드하여 직접 플레이 해볼 수 있습니다.<br><br>
 전체 플레이하기 ↓ <br>
 링크<br>
<br><br><br>
 아래 링크을 클릭하면 유튜브 링크로 이동해 플레이 및 설명하는 동영상을 시청할 수 있습니다. <br>
